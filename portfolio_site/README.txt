Personal portfolio site - single page (static)
Files:
- index.html : main portfolio page (responsive, light/dark toggle)
- styles.css : styling
- app.js : small interactive scripts (theme toggle + resume generator)
- profile.jpg : placeholder image (replace with your photo)
- proj-*.png : placeholder project images (replace with screenshots)
How to use:
1. Unzip the folder.
2. Replace profile.jpg and project images with your own images.
3. Edit index.html to update text (projects, about, skills).
4. Open index.html in a browser or use Live Server in VS Code.
